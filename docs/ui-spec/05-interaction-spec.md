# 05. 全局交互规范

> **用途**: 跨组件统一的交互行为定义，确保整个产品体验一致。  
> **适用**: 所有基于 `@company/ui` 组件库构建的项目。

---

## 5.1 动画系统（Motion）

### 5.1.1 缓动函数（Easing）

| Token 名称 | 值 | 用途 |
|-----------|-----|------|
| `--ease-out` | `cubic-bezier(0.215, 0.61, 0.355, 1)` | 元素出现、展开（先快后慢） |
| `--ease-in` | `cubic-bezier(0.55, 0.055, 0.675, 0.19)` | 元素消失、收起（先慢后快） |
| `--ease-in-out` | `cubic-bezier(0.645, 0.045, 0.355, 1)` | 位置移动、样式过渡 |
| `--ease-bounce` | `cubic-bezier(0.175, 0.885, 0.32, 1.275)` | 弹窗出现（微弹性） |

### 5.1.2 动画时长

| Token 名称 | 值 | 用途 |
|-----------|-----|------|
| `--duration-fast` | `100ms` | 微交互：hover 变色、checkbox 切换 |
| `--duration-base` | `200ms` | 标准过渡：展开/收起、fade in/out |
| `--duration-slow` | `300ms` | 大幅度动画：弹窗出现、抽屉滑入、页面切换 |
| `--duration-ultra-slow` | `500ms` | 复杂编排：骨架屏闪烁、进度条动画 |

### 5.1.3 各组件动画速查

| 交互 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 按钮 hover | 背景色渐变 | `--duration-fast` | `ease` |
| 输入框 focus | 边框变色 + 阴影出现 | `--duration-base` | `--ease-out` |
| 下拉菜单展开 | 透明度 0→1 + 上移 4px | `--duration-base` | `--ease-out` |
| 下拉菜单收起 | 透明度 1→0 + 下移 4px | `--duration-base` | `--ease-in` |
| 弹窗打开 | 透明度 0→1 + 缩放 0.95→1 | `--duration-slow` | `--ease-bounce` |
| 弹窗关闭 | 透明度 1→0 + 缩放 1→0.95 | `--duration-base` | `--ease-in` |
| 抽屉打开 | 从右/左/上/下滑入 | `--duration-slow` | `--ease-out` |
| 抽屉关闭 | 滑出 | `--duration-base` | `--ease-in` |
| 气泡/Tooltip | 透明度 0→1 | `150ms` | `--ease-out` |
| 消息提示出现 | 上滑 8px + 透明度 0→1 | `--duration-base` | `--ease-out` |
| 消息提示消失 | 上滑 8px + 透明度 1→0 | `--duration-base` | `--ease-in` |
| 表格行 hover | 背景色渐变 | `--duration-fast` | `ease` |
| 表格排序图标 | 颜色渐变 | `--duration-fast` | `ease` |
| 标签(Tag)删除 | 缩放 1→0 + 透明度 1→0 | `--duration-base` | `--ease-in` |
| Tab 切换下划线 | 水平位移 | `--duration-slow` | `--ease-in-out` |
| 骨架屏 | 渐变光从左到右移动 | `1.4s` | `ease-in-out` (infinite) |

## 5.2 反馈系统（Feedback）

### 5.2.1 Loading（加载态）

| 场景 | 加载方式 | 位置 | 说明 |
|------|---------|------|------|
| **页面首次加载** | 骨架屏 (Skeleton) | 内容区域 | 模拟页面结构，避免白屏 |
| **数据刷新（已知结构）** | Spin + 半透明遮罩 | 覆盖内容区 | 透明度 `0.6`，保留表头可见 |
| **按钮操作中** | 按钮内 Loading 图标 + 禁用 | 按钮左侧 | 图标替换原有 icon，文字不变 |
| **下拉加载更多** | 底部 Spin | 列表底部 | 小号加载图标 |
| **图片加载** | 灰色占位块 → 图片淡入 | 图片位置 | 灰色块尺寸与图片一致 |
| **后台静默刷新** | 顶部细进度条 | 页面最顶部 | 2px 高度，主色 |

### 5.2.2 Empty（空状态）

```
┌──────────────────────────────────┐
│                                  │
│         [插画/图标]               │  ← 灰色调插画，尺寸 120×120px
│                                  │
│        暂无数据                   │  ← --color-text-secondary, 14px
│   点击下方按钮新建第一条记录       │  ← --color-text-tertiary, 13px
│                                  │
│       [ 新建记录 ]               │  ← 可选的操作引导按钮
│                                  │
└──────────────────────────────────┘
```

| 场景 | 插画/图标 | 主文案 | 副文案 | 操作按钮 |
|------|----------|--------|--------|---------|
| 列表无数据 | 空盒子插画 | `暂无数据` | — | — |
| 搜索无结果 | 搜索插画 | `未找到符合条件的结果` | `请尝试调整筛选条件` | `清除条件` |
| 消息/通知为空 | 铃铛插画 | `暂无消息` | — | — |
| 权限不足 | 锁插画 | `暂无访问权限` | `请联系管理员开通权限` | — |
| 接口报错 | 感叹号插画 | `数据加载失败` | `请检查网络后重试` | `重新加载` |

### 5.2.3 Error（错误态）

#### 表单校验错误

| 元素 | 规范 |
|------|------|
| 错误输入框边框 | `--color-danger` + `box-shadow: 0 0 0 2px rgba(255,77,79,0.2)` |
| 错误提示文字 | `--color-danger`, `--font-size-xs` (`12px`), 显示在输入框下方 |
| 错误图标 | `CloseCircleFilled`, `14px`, `--color-danger` |
| 校验时机 | 失焦时触发（onBlur） |

#### 页面级错误

| 场景 | 处理方式 |
|------|---------|
| 接口 500 | 显示错误提示 Message + 保留当前页面数据 |
| 接口 403 | 跳转 403 页面 或 展示无权限空状态 |
| 接口超时 | Message 提示"请求超时，请重试" + 保留操作数据 |
| 表单提交失败 | Message.error 提示具体错误原因 |

### 5.2.4 Success（成功态）

| 场景 | 反馈方式 | 时长 |
|------|---------|------|
| 新增成功 | Message.success + 关闭弹窗 + 刷新列表 | 3s 自动消失 |
| 删除成功 | Message.success + 刷新列表 | 3s |
| 编辑成功 | Message.success | 3s |
| 批量操作成功 | Message.success + 清除选中项 | 3s |
| 文件上传成功 | 文件列表中状态变为"已上传" | — |

### 5.2.5 Confirm（确认）

| 场景 | 确认方式 | 标题 | 内容 |
|------|---------|------|------|
| 删除单条 | Popconfirm（气泡确认） | `确定要删除吗？` | `删除后不可恢复` |
| 批量删除 | Modal.confirm | `确定要批量删除吗？` | `已选择 N 条记录，删除后不可恢复` |
| 离开未保存 | Modal.confirm | `有未保存的更改` | `确定要离开此页面吗？未保存的内容将丢失` |
| 危险操作 | Modal.confirm（红色确认按钮） | `确定要执行此操作吗？` | `此操作不可撤销` |

## 5.3 响应式断点

| 断点名称 | 宽度 | 列数 | 说明 |
|---------|------|------|------|
| `xs` | `< 576px` | 1 列 | 手机 |
| `sm` | `≥ 576px` | 2 列 | 大屏手机/小平板 |
| `md` | `≥ 768px` | 3 列 | 平板横屏 |
| `lg` | `≥ 992px` | 3~4 列 | 小屏笔记本 |
| `xl` | `≥ 1200px` | 4 列 | 标准桌面 |
| `xxl` | `≥ 1600px` | 4~6 列 | 大屏显示器 |

### 组件行为变化

| 组件 | `< 768px` | `≥ 768px` |
|------|-----------|-----------|
| Table | 转为卡片列表，每行一个卡片 | 标准表格 |
| Form | 表单项全宽，label 在顶部 | 表单项按栅格排列，label 在左侧 |
| Breadcrumb | 隐藏，改为"返回上级"按钮 | 完整显示 |
| Modal | 全屏弹窗 | 居中弹窗，最大宽度 `520px`/`720px`/`960px` |
| Drawer | 宽度 `100%` | 宽度 `380px`/`520px`/`720px` |
| Button 组 | 垂直排列 | 水平排列 |

## 5.4 键盘交互规范

| 组件 | 快捷键 | 行为 |
|------|--------|------|
| Modal | `Esc` | 关闭弹窗（如果 `maskClosable` 为 true） |
| Modal | `Enter` | 触发确认按钮（如果有） |
| Select（下拉） | `↑ ↓` | 选项间切换 |
| Select（下拉） | `Enter` | 选中高亮选项 |
| Select（下拉） | `Esc` | 关闭下拉面板 |
| Table（可编辑） | `Tab` | 移动到下一个可编辑单元格 |
| Table（可编辑） | `Enter` | 进入/退出编辑态 |
| Table（可编辑） | `Esc` | 取消编辑，恢复原值 |
| DatePicker | `Esc` | 关闭日期面板 |
| 全局 | `Ctrl+S` / `Cmd+S` | 保存当前表单（需业务层实现） |
| 搜索框 | `Enter` | 触发搜索 |

## 5.5 数据格式规范

| 数据类型 | 显示格式 | 示例 | 空值显示 |
|---------|---------|------|---------|
| 日期 | `YYYY-MM-DD` | `2024-01-15` | `--` |
| 日期时间 | `YYYY-MM-DD HH:mm:ss` | `2024-01-15 14:30:00` | `--` |
| 金额 | ¥ 分隔，保留 2 位 | `¥1,234.56` | `--` |
| 百分比 | 保留 1 位小数 + `%` | `85.5%` | `--` |
| 长数字 | 千分位分隔 | `1,234,567` | `--` |
| 手机号 | 前三后四脱敏 | `138****5678` | `--` |
| 布尔值 | `是` / `否` | — | `--` |
| null/undefined | 统一显示 | `--` | `--` |
| 空字符串 | 同 null | `--` | `--` |

## 5.6 可访问性（A11y）最低要求

| 要求 | 标准 | 说明 |
|------|------|------|
| 颜色对比度 | WCAG AA (4.5:1 普通文本, 3:1 大文本) | 暗色主题尤其需要验证 |
| 焦点指示器 | `:focus-visible` 所有可交互元素 | 默认使用 `--color-primary` 2px outline |
| 键盘可操作 | 所有交互可用 Tab/Enter/Esc 完成 | 见 5.4 键盘规范 |
| `aria-label` | 图标按钮必须有 | 如 `aria-label="删除"` |
| `role` | 语义化 role 属性 | Table 用 `role="table"`, Dialog 用 `role="dialog"` |
