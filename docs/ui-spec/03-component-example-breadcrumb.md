# 03. 面包屑 Breadcrumb（完整示例）

> **组件路径**: `@company/ui/Breadcrumb`  
> **底层依赖**: ant-design-vue `a-breadcrumb`  
> **参照**: [飞书开放平台 - Breadcrumb](https://open.feishu.cn/document/design-specification/component---navigation/breadcrumb)  
> **设计稿 Figma 链接**: `[待 UI 补充]`

---

## 一、使用规则

| 使用场景 | 不使用场景 |
|----------|-----------|
| 系统层级超过 2 级以上时使用 | 系统只有一层，无需层级导航 |
| 需要告知用户当前所处位置 | 页面标题已经足够表达当前位置 |
| 需要提供向上导航，快速回到上级页面 | 不需要返回上级（如独立的详情页） |
| — | 不要作为页面主标题使用 |

---

## 二、组成要素

```
┌──────────────────────────────────────────────────────┐
│  🏠  首页  >  数据管理  >  ···  >  数据中心  >  详情   │
│  ↑    ↑       ↑          ↑        ↑          ↑       │
│  首页  文本    分割符     省略符    当前层级    最后一项  │
│  图标                                      不可点击   │
└──────────────────────────────────────────────────────┘
```

| 序号 | 要素名称 | 说明 | 是否必选 |
|------|---------|------|---------|
| 1 | **首页图标** | 固定显示在最左侧，点击回到首页 | 必选 |
| 2 | **层级文本** | 每级页面的名称，除最后一级外均可点击 | 必选 |
| 3 | **分割符** | 分隔层级，使用 `>` 或 `/` 符号 | 必选 |
| 4 | **省略符**（`···`） | 层级过多时折叠中间层级，hover 气泡展示完整路径 | 条件必选（超宽时出现） |
| 5 | **当前层级** | 最后一级，不可点击，字重加粗 | 必选 |
| 6 | **容器背景** | 承载折叠内容的背景面板（当层级过多时） | 条件必选 |

## 三、尺寸规格

| 尺寸 | Token | 高度 | 字号 | 图标大小 | 适用场景 |
|------|-------|------|------|---------|---------|
| Default | — | `22px` (行高) | `--font-size-sm` (`13px`) | `14px` | **标准**，所有页面 |

> 面包屑只有一种尺寸规格，不做 Small / Large 变体。

## 四、状态矩阵（⚠️ 核心交付物）

### 4.1 层级文本状态

| 状态 | 文字色 | 背景色 | 字重 | 光标 | 下划线 |
|------|--------|--------|------|------|--------|
| **Default**（可点击层级） | `--color-text-secondary` (`#595959`) | 透明 | `--font-weight-regular` (`400`) | `pointer` | 无 |
| **Hover**（悬浮） | `--color-primary` (`#2575F5`) | 透明 | `--font-weight-regular` | `pointer` | 无 |
| **Active**（点击） | `--color-primary-active` (`#1A5FCF`) | 透明 | `--font-weight-regular` | `pointer` | 无 |
| **Current**（当前层级-最后一项） | `--color-text-primary` (`#262626`) | 透明 | `--font-weight-semibold` (`600`) | `default` | 无 |
| **Disabled**（禁用层级） | `--color-text-disabled` (`#D9D9D9`) | 透明 | `--font-weight-regular` | `not-allowed` | 无 |

### 4.2 省略符状态

| 状态 | 文字色 | 图标 | 光标 | 气泡 |
|------|--------|------|------|------|
| **Default** | `--color-text-secondary` | `···` 或 `MoreOutlined` | `pointer` | — |
| **Hover** | `--color-primary` | 同 Default | `pointer` | 弹出 `--shadow-base`，展示折叠的全部层级 |

### 4.3 分割符

| 属性 | 值 |
|------|-----|
| 图标 | `>` (默认) 或 `/` |
| 颜色 | `--color-text-tertiary` (`#BFBFBF`) |
| 尺寸 | `14px` |
| 与两侧文本间距 | `--spacing-xs` (`4px`) |

### 4.4 首页图标

| 状态 | 颜色 | 尺寸 |
|------|------|------|
| **Default** | `--color-text-secondary` | `14px` |
| **Hover** | `--color-primary` | `14px` |

## 六、超长 / 边界情况处理

| 场景 | 处理规则 | 效果描述 |
|------|---------|---------|
| **路径超过容器宽度** | 从第 2 层级开始折叠，用 `···` 替代，hover 弹出气泡展示完整路径 | 面包屑不超过内容区宽度的 **1/2**，超宽则触发折叠 |
| **极端超长 - 只剩 2 层级也放不下** | 优先展示第 1 级和最后 1 级 | 格式：`首页 > ··· > 最后一级` |
| **单层级文本超长** | 单层最大宽度 `162px`（约 10 个中文字符），超出省略 | 省略部分 hover 气泡展示 |
| **路径只有 1 级** | 只显示当前层级，不显示首页图标和分割符 | `当前页面名称` |
| **路径只有 2 级** | 不显示省略符 | `首页 > 当前页` |
| **页面宽度 < 768px** | 仅显示"返回上一级"按钮，不展示完整面包屑 | 节约移动端空间 |

## 十、前端实现注意事项

| 序号 | 注意事项 |
|------|---------|
| 1 | ant-design-vue 的 `a-breadcrumb` 默认分割符是 `/`，需要改为 `>` 需通过 `separator` prop |
| 2 | 省略符折叠逻辑需前端自行实现（`a-breadcrumb` 不内置），建议封装 `useBreadcrumbCollapse` hook |
| 3 | 暗色主题下 `--color-text-secondary` 的 `#A6A6A6` 在深色背景上对比度约 4.2:1，刚好通过 WCAG AA 标准 |
| 4 | 首页图标建议使用 slot 方式传入，不耦合具体图标库 |
